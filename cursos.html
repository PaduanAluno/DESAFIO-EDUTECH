Guia para o aluno edutech

E é aqui que falaremos um pouquinho sobre cada curso. Ele terá uma certa ordem que você irá aprender, então vamos começar!

1º Lógica de programação

Aqui é onde tudo começa. Aqui, você irá aprender o básico da programação usando o JavaScript, que mais para frente será mais apronfundado. Basicamente, você entenderá elementos simples porém importantes como as variáveis (var), as séries de if's, else if e else, os laços com while e for e muito mais! Com isso será possível fazer lógicas como a de operações matemáticas, fazer repetições, contagens e etc.

2º HTML5 e CSS3

Assim como no curso anterior em que a programação começa com a lógica, para a criação de sites temos que aprender dentre todas as coisas, a usar o HTML e o CSS, na trilha front-end vocês terão 4 cursos específicos para essa parte. Só para você terem uma ideia do que se trata essas duas siglas, o HTML serve para criarmos uma estrutura da nossa página, como se fossem os ossos de nosso corpo, já o CSS serve para estilizarmos nosso site, como se fossem nossa pele, nosso cabelo, etc. Então, aqui você aprenderá a usar o HTML e o CSS para criarmos e estilizarmos uma página da web do ZERO!

3º Flexbox

Os dois primeiros cursos tratam das bases do mundo front-end. Agora, a partir de agora, vamos dar uma avançada e ver mais a fundo elementos desse mundo que nos ajudam a melhorar os nossos sites e ainda ajudar a nós mesmos a programar. Aqui, temos o Flexbox, que serve justamente para melhorar e facilitar o manuseio dos elementos do HTML, podendo ajustar seu site de maneira simples.

4º Layoults Responsivos

E sobre aprofundarmos no front-end, não adianta nada saber fazer sites, mas fazermos apenas para os dispositivos de tela grande, temos que ter uma responsividade, uma adaptação para dispositivos de telas menores como os tão presentes celulares. Então nesse curso aprenderemos como criar sites para as mais variadas telas dos mais variados dispositivos!

5º CSS Grid Layoult

Assim como o flexbox que nos ajuda no manejamento dos elementos do HTML, o grid nos ajuda da mesma forma, porém aqui, podemos criar como se fossem bloquinhos dentro de um bloco maior, que poderemos ajustar da maneira que quisermos deixando um bloquinho maior do que o outro, ou deixarmos todos com um tamanho proporcional. Assim, podemos fazer designs muito mais complexos e responsivos.

6º Arquitetura CSS

Bem, a arquitetura css, seria a maneira como organizaríamos os nossos arquivos css e o que escrevemos. Para isso, há várias metodologias e boas práticas feitas no CSS que serão ditas ao decorrer do curso. Uma dessas metodologias será a Atomic Design que na prática se divide em cinco estágios: átomos, moléculas, organismos, modelos e paginas.

7º Bootstrap

Aqui temos um framework web, isso mesmo, um nome meio diferente certo? Pois bem o bootstrap serve para termos acesso a componentes de interface e front-end para sites e aplicações web usando HTML, CSS e JavaScript já prontos para uso. Como exemplo, você poderia pegar um código de um carrousel no bootsrap e incorporar no seu site com facilidade e rapidez e melhor, responsivo!
