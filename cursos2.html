Continuando com os cursos..

8º Guia de estilos

Nesse curso aprenderemos sobre como usar ferramentas de layoult como o FIGMA - que é usada nesse curso - e assim conseguir analisar um layoult e retirar todos os elementos que foram inseridos, para passar para o código e assim construir o site proposto. Além de tudo isso, revemos algumas boas práticas do CSS e organizar todos os arquivos e códigos CSS.

9º JavaScript

Lembra dos cursos de lógica de programação? Aqui você conseguirá aprender e ficar mais imerso no mundo do JavaScript, a programação no mundo do front-end. Então nesse curso você aprenderá diversas coisas sobre o JS especificamente e suas particularidades em relação a outras linguagens como Python ou C#, mas isso fica para outra trilha ;)

10º Acessibilidade

Pensamos na responsividade para atingirmos um maior número de pessoas, então porque não tornarmos os nosso sites mais acessíveis? Deixá-lo viável para cada tipo de pessoa que existe independente de suas dificuldades? É para isso que esse curso existe! Aqui você conseguirá aprender boas e simples práticas para facilitar para todo mundo o acesso aos sites que você cria.

11º HTTP

Já aqui nós mergulhamos mais dentro da internet e aprendemos diversos conceitos e particularidades desse mundo. Então, deixamos de lado um pouco o código real, para pensarmos em como as coisas na internet funcionam, a ligação entre front-end e back-end e todas as coisas que essa relação geram.

12º Vetores e animação com SVG

Você sabia que podemos incorporar ou ATÉ MESMO criar animações com JavaScript e o CSS? Pois é, é possível. Então aqui você aprenderá maneiras de criar ou incorporar animações já existentes para assim deixar o seu site mais interessante e bonito.
