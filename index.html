Guia para o aluno edutech

O aluno EDUTECH pode aprender a criar um mar de possibilidades no front-end com base nos cursos que a ALURA disponibiliza para ele.

Mas o que é o programa EDUTECH?

O EDUTECH é um programa desenvolvido pelo governo do Estado em parceria com a Alura, isso mesmo! Com isso foram criadas trilhas específicas para cada tipo de área do mundo da programação para nós estudantes aprendermos desde a criar jogos, para até mesmo criar aqueles sites bonitões que vemos por aí.

Com isso, esse site tem como objetivo ajudar o jovem estudante em sua jornada pelo mundo do front-end. Iremos trazer uma visão geral dos cursos oferecidos aos alunos para assim você não ficar perdido nesse imenso mundo!
